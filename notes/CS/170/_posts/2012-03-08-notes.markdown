### Without Repetitions

    K(w, i) = the highest value I can pack in a knapsack of wt w when the
              available items are 1 .. i

    K(0, i) = K(w, 0) = 0
    for i = 1 to n:
      for w = 1 to w:
        if w > n K(w, i) = K(w, i-1)
        else: K(w, i) = max[K(w, i-1), K(w-wi, i-1)] + vi
